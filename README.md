# podcast_tools

A python-based tool suite for podcast. Tools include:
1. creating audiogram:
    - creates an audiogram from audio input for YouTube
    - runs in parallel using `multiprocessing`

## Dependencies
python libraries:
- `matplotlib`
- `scipy`
- `numpy`
- `moveipy`
- `pydub`
- `mutagen`
\\
Additional:
- ffmpeg
